\title{visStatistics: A transparent visual decision pipeline for statistical hypothesis tests in R}


\author{by Sabine Schilling}

\maketitle

\abstract{%
We present \CRANpkg{visStatistics}, an R package that implements a transparent decision pipeline for selecting, executing, and visualising common statistical hypothesis tests between two vectors. Given vectors of class \code{numeric}, \code{integer}, or \code{factor}, the main function \code{visstat()} routes the analysis according to variable types, sample size, and displayed assumption diagnostics. Visual outputs --- box plots, bar charts, regression lines with confidence bands, mosaic plots, and residual diagnostics --- are annotated with test statistics, assumption-check results, and post-hoc comparisons where applicable. The decision logic follows the general linear model framework, assessing normality on pooled model residuals rather than per group, which preserves statistical power. The package is particularly suited for server-side R applications where end users interact exclusively via a web interface, and for time-constrained settings such as statistical consulting or introductory teaching. All tests are applied using their default settings from base R, keeping the dependency footprint minimal and the decision tree reproducible.
}

\section{Introduction}\label{introduction}

Most routine data analyses reduce to a comparatively small set of
inferential frameworks: group comparisons, contingency-table analyses,
correlation, and simple regression \citep{Sato:2017, Chicco:2025}. Selecting
the appropriate test within each framework, however, requires evaluating
variable types, sample sizes, and distributional assumptions --- a
process that is error-prone, time-consuming, and a recurring bottleneck
in statistical consulting and undergraduate teaching.

Several R packages address parts of this problem.
\CRANpkg{compareGroups} \citep{Subirana:2014} automates test choice via a
per-group Shapiro--Wilk normality test and produces publication-ready
descriptive tables, but its primary output is tabular rather than
graphical, and assumption diagnostics are not displayed alongside
results. \CRANpkg{ggstatsplot} \citep{Patil:2021} produces information-rich
\CRANpkg{ggplot2}-based \citep{Wickham:2016} plots with embedded test
results, but requires the user to select the appropriate plotting
function for each data-type combination (e.g., \texttt{ggbetweenstats()} for
group comparisons, \texttt{ggscatterstats()} for regression). \CRANpkg{rstatix}
\citep{Kassambara:2025} provides a pipe-friendly framework for common tests
with tidy output, and \CRANpkg{ggpubr} \citep{Kassambara:2026} adds
statistical annotations to publication-ready plots; both leave test
selection entirely to the user.

\CRANpkg{visStatistics} takes a different approach: a single function
\texttt{visstat(x,\ y)} accepts any combination of numeric and categorical
vectors and resolves the full decision pipeline --- test selection,
execution, assumption checking, and visualisation --- without any
further input. Three design choices distinguish it from the packages
above.

The contribution is not a new hypothesis test, but a reproducible
software workflow around existing tests. \texttt{visstat()} combines a unified
two-vector interface, explicit routing by input class and diagnostics,
graphical assumption checks shown alongside the selected result, and
branch-specific post-hoc procedures. The aim is to make routine default
analyses transparent and inspectable, not to replace statistical
judgement in cases where the displayed diagnostics suggest that another
model is more appropriate.

First, normality is assessed on the pooled residuals of a linear model
\texttt{lm(y\ \textasciitilde{}\ x)}, following the general linear model (GLM) framework
\citep{Searle:1971}, rather than per group. Group-wise testing is
statistically less powerful and more likely to reject normality
unnecessarily, pushing the analysis to non-parametric methods
\citep{Lumley:2002}. When heteroscedasticity is subsequently detected,
group-wise normality diagnostics are shown additionally, since Welch's
methods use group-specific variance estimates.

Second, assumption diagnostics are always displayed, regardless of
whether assumptions are met. This enables users to visually override the
automated decision in individual cases, and supports the pedagogical
goal of making assumption checking an integral part of the analysis
rather than an afterthought.

Third, all tests use their default settings from base R functions,
keeping the dependency footprint minimal and the decision tree
transparent. The package does not wrap external statistical engines; the
automated layer is purely in the routing logic.

\section{Package overview}\label{package-overview}

\CRANpkg{visStatistics} is available on CRAN. The package accepts input
in two equivalent forms:

\begin{verbatim}
# Recommended form:
visstat(x, y)

# Formula interface:
visstat(y ~ x, data = dataframe)
\end{verbatim}

where \texttt{x} and \texttt{y} are vectors of class \texttt{"numeric"}, \texttt{"integer"}, or
\texttt{"factor"}. The function returns an object of class \texttt{"visstat"} with
\texttt{print()}, \texttt{summary()}, and \texttt{plot()} S3 methods, and optionally writes
graphics to disk via \CRANpkg{Cairo} \citep{Urbanek:2025}.

\section{Decision logic}\label{decision-logic}

The significance level \(\alpha = 1 - \texttt{conf.level}\) is
user-controlled (default 0.95). Figure \ref{fig:overview} gives a
high-level overview of all implemented tests by input class; Figure
\ref{fig:decision-tree} details the decision path for a numeric
response and categorical predictor. Technical definitions of all test
statistics are collected in \hyperref[tests]{Appendix B}.

\begin{figure}
\includegraphics[width=1\linewidth,alt={Flowchart showing all implemented statistical tests organised by the class of the input vectors.}]{figures/overview} \caption{Overview of all implemented tests by input class.}\label{fig:overview}
\end{figure}

\begin{figure}
\includegraphics[width=1\linewidth,alt={Decision tree selecting among Welch t-test, Student t-test, Wilcoxon, Fisher ANOVA, Welch ANOVA, and Kruskal-Wallis tests based on sample size, Shapiro-Wilk, and Levene test results.}]{figures/decision_tree} \caption{Decision tree for a numeric response and categorical predictor, based on sample size, normality of residuals, and homoscedasticity.}\label{fig:decision-tree}
\end{figure}

Figure \ref{fig:overview} is read by input class. Numeric responses
with categorical predictors enter the central-tendency branch; two
unordered factors enter the proportion-comparison branch; two numeric
variables enter simple linear regression unless \texttt{correlation\ =\ TRUE},
which switches to Spearman rank correlation; and two ordered factors can
be analysed as Kendall rank correlation when \texttt{correlation\ =\ TRUE}.
Figure \ref{fig:decision-tree} then expands the central-tendency
branch. Following the linear-model framing introduced above and detailed
in \hyperref[glm]{Appendix A}, the first split is standardised residual
normality. If Shapiro--Wilk rejects residual normality, \texttt{visstat()}
selects the non-parametric path (\texttt{wilcox.test()} for two groups,
\texttt{kruskal.test()} for more than two groups, with pairwise Wilcoxon
post-hoc comparisons. If residual normality is not rejected,
Levene--Brown--Forsythe decides between the equal-variance parametric
path (\texttt{t.test(var.equal\ =\ TRUE)} or \texttt{aov()} with \texttt{TukeyHSD()}) and the
heteroscedastic Welch path (\texttt{t.test()} or \texttt{oneway.test()} with
\texttt{games.howell()}).

\subsection{Numeric response and categorical predictor: comparing central tendencies}\label{numeric-response-and-categorical-predictor-comparing-central-tendencies}

A linear model \texttt{lm(y\ \textasciitilde{}\ x)} is fitted and assumption diagnostics are
always displayed via \texttt{vis\_lm\_assumptions()} (see \hyperref[glm]{Appendix A}).
The Shapiro--Wilk test \citep{Shapiro:1965} is applied to internally
studentised residuals from \texttt{rstandard()} \citep{Cook:1982}. When all groups
contain more than 50 observations, normality is assumed by the central
limit theorem \citep{Lumley:2002, Rasch:2011}.

If normality is rejected (\(p_\text{SW} \le \alpha\)), non-parametric
tests are selected: \texttt{wilcox.test()} \citep{Mann:1947} for two groups, or
\texttt{kruskal.test()} \citep{Kruskal:1952} followed by Holm-adjusted
\texttt{pairwise.wilcox.test()} \citep{Holm:1979} for more than two groups.

If normality is not rejected, the Levene--Brown--Forsythe test
\texttt{levene.test()} \citep{Brown:1974} (see \hyperref[tests]{Appendix B}) assesses
variance homogeneity. For homoscedastic data, Student's
\texttt{t.test(var.equal\ =\ TRUE)} or Fisher's \texttt{aov()} \citep{Fisher:1990} with
\texttt{TukeyHSD()} \citep{Hochberg:1987} is applied. For heteroscedastic data,
Welch's \texttt{t.test()} \citep{Welch:1947, Satterthwaite:1946} or \texttt{oneway.test()}
\citep{Welch:1951} with \texttt{games.howell()} \citep{Games:1976} is applied. When
switching to Welch methods, group-wise normality diagnostics are
additionally displayed via \texttt{vis\_group\_normality()}, since the pooled
\texttt{lm()} residuals are no longer the appropriate diagnostic under
heteroscedasticity.

\subsection{Both variables numeric: regression or Spearman rank correlation}\label{both-variables-numeric-regression-or-spearman-rank-correlation}

By default, \texttt{visstat()} fits a simple linear regression via \texttt{lm()},
regardless of whether GLM assumptions are met, and always displays
assumption diagnostics. When \texttt{correlation\ =\ TRUE} is set explicitly,
Spearman's \(\rho\) is computed via \texttt{cor.test(...,\ method\ =\ "spearman")}.
Because the choice between modelling a directional relationship and
measuring a monotone association cannot be automated from data types
alone, Spearman correlation is never triggered automatically.

\subsection{Both variables categorical: comparing proportions}\label{both-variables-categorical-comparing-proportions}

When both variables are categorical, \texttt{visstat()} tests independence
using Pearson's \(\chi^2\) test (\texttt{chisq.test()}) or Fisher's exact test
(\texttt{fisher.test()}), depending on expected cell counts following Cochran's
rule \citep{Cochran:1954}: the \(\chi^2\) approximation is used if no expected
cell count is less than 1 and no more than 20\% of cells have expected
counts below 5. Yates' continuity correction \citep{Yates:1934} is applied by
default to \(2 \times 2\) tables. For general \(R \times C\) tables,
Pearson's \(\chi^2\) result is supplemented with a mosaic plot from
\CRANpkg{vcd} \citep{Meyer:2006, Meyer:2024} with tiles coloured by Pearson
residuals.

\subsection{Ordered factors}\label{ordered-factors}

When the response is an ordered factor, \texttt{visstat()} converts it to
integer level codes and follows the non-parametric path. When both
variables are ordered and \texttt{correlation\ =\ TRUE}, Kendall's \(\tau_b\)
\citep{Kendall:1945, Agresti:2010} is computed instead, as it corrects for
ties explicitly --- unavoidable with few ordinal levels \citep{Xu:2013}.

\section{Examples}\label{examples}

The examples follow the same order as the decision logic. The number of
plots returned by \texttt{visstat()} depends on the input types and the outcome
of assumption checks. Most tests produce two panels: an
assumption-diagnostic panel and a result panel. When heteroscedasticity
is detected and \texttt{visstat()} routes to a Welch method, a third panel with
group-wise normality diagnostics is added, since pooled-residual
diagnostics are no longer appropriate under unequal variances. Tests
that have no separate assumption diagnostics, such as Kendall's \(\tau_b\)
for two ordered factors, produce a single plot.

The datasets were chosen to exercise the main branches of the decision
tree: normal and homoscedastic residuals leading to Student's t-test or
one-way ANOVA; normal residuals with unequal variances leading to
Welch's t-test or Welch's one-way test; non-normal residuals leading to
Wilcoxon or Kruskal--Wallis tests, with Holm-adjusted pairwise Wilcoxon
tests for the multi-group extension; numeric--numeric regression and
rank correlation; categorical independence tests under and beyond
Cochran's rule; and ordinal rank correlation.

\subsection{Numeric response and categorical predictor: two groups}\label{numeric-response-and-categorical-predictor-two-groups}

For two groups, \texttt{visstat()} first evaluates the linear-model assumptions
on pooled residuals. If Shapiro--Wilk rejects normality, the decision
path switches to the Wilcoxon rank-sum test. If normality is not
rejected, the Levene--Brown--Forsythe test decides between Student's
equal-variance t-test and Welch's unequal-variance t-test.

\subsubsection{Welch's t-test}\label{welchs-t-test}

The \emph{Motor Trend Car Road Tests} dataset (\texttt{mtcars}) contains 32
observations, where \texttt{mpg} denotes miles per (US) gallon and \texttt{am}
represents the transmission type (\texttt{0} = automatic, \texttt{1} = manual). With
binary factor \texttt{am} and continuous response \texttt{mpg}, Shapiro--Wilk does not
reject normality of the pooled residuals, while the
Levene--Brown--Forsythe test detects heteroscedasticity. The routing
therefore leads to Welch's t-test rather than Student's t-test.

\begin{verbatim}
mtcars$am <- as.factor(mtcars$am)
t_test_stats <- visstat(mtcars$am, mtcars$mpg)
\end{verbatim}

\subsubsection{Wilcoxon rank-sum test}\label{wilcoxon-rank-sum-test}

The Wilcoxon branch is exemplified on examination grades for a class of
44 students (21 girls, 23 boys). After fitting \texttt{lm(grade\ \textasciitilde{}\ sex)},
Shapiro--Wilk rejects normality of the pooled residuals, so \texttt{visstat()}
switches to the non-parametric Wilcoxon rank-sum test.

\begin{verbatim}
grades_gender <- data.frame(
  sex   = as.factor(c(rep("girl", 21), rep("boy", 23))),
  grade = c(19.3, 18.1, 15.2, 18.3,  7.9,  6.2, 19.4, 20.3,  9.3, 11.3,
            18.2, 17.5, 10.2, 20.1, 13.3, 17.2, 15.1, 16.2, 17.0, 16.5,
             5.1, 15.3, 17.1, 14.8, 15.4, 14.4,  7.5, 15.5,  6.0, 17.4,
             7.3, 14.3, 13.5,  8.0, 19.5, 13.4, 17.9, 17.7, 16.4, 15.6,
            17.3, 19.9,  4.4,  2.1)
)
wilcoxon_stats <- visstat(grades_gender$sex, grades_gender$grade)
\end{verbatim}

\end{figure}

\subsection{Numeric response and categorical predictor: more than two groups}\label{numeric-response-and-categorical-predictor-more-than-two-groups}

For more than two groups, the same residual-normality decision is
applied first. If residual normality is rejected, \texttt{visstat()} applies
\texttt{kruskal.test()}. Otherwise, \texttt{levene.test()} chooses between Fisher's
one-way ANOVA (\texttt{aov()}) and Welch's heteroscedastic one-way ANOVA
(\texttt{oneway.test()}). The graphical output consists of two panels. The
first panel displays standardised residuals versus fitted values and a
normal Q--Q plot; no more than about 5\% of the standardised residuals
should exceed \(|2|\) when residuals are normally distributed, and the
Q--Q points should approximately follow the red reference line. The
title reports the Shapiro--Wilk and Anderson--Darling normality tests
together with Bartlett's and Levene--Brown--Forsythe tests for
homoscedasticity. The second panel shows the selected omnibus test as
box plots with sample sizes above each group; for the parametric
branches, the title also reports the corresponding \(F\) statistic. When
the largest group has no more than 50 observations, jittered points are
overlaid to show the raw data.

ANOVA, Welch ANOVA, and Kruskal--Wallis are omnibus tests: a significant
result tells us that \emph{some} group differs, but not which. To identify
the differing pairs, \texttt{visstat()} tests all pairwise comparisons among
the factor levels, defining a family of tests \citep{Abdi:2007}. Because the
three omnibus tests rest on different assumptions, each branch uses a
matching post-hoc procedure: \texttt{TukeyHSD()} after \texttt{aov()},
\texttt{games.howell()} after \texttt{oneway.test()}, and
\texttt{pairwise.wilcox.test(p.adjust.method\ =\ "holm")} after \texttt{kruskal.test()}.
\texttt{TukeyHSD()} controls the family-wise error rate through the studentised
range distribution under a common-variance assumption. \texttt{games.howell()}
uses separate variance estimates and Welch-adjusted degrees of freedom
for each pair, making it the post-hoc procedure for the heteroscedastic
Welch branch. The Kruskal--Wallis branch uses Holm's step-down
adjustment for the pairwise Wilcoxon tests. Pairs whose adjusted
post-hoc \(p\)-value falls below \(\alpha\) are marked with different green
letters below the box plots; pairs sharing a letter are not
significantly different. The returned object contains the corresponding
adjusted \(p\)-values.

\subsubsection{One-way ANOVA with Tukey HSD}\label{one-way-anova-with-tukey-hsd}

The \texttt{npk} dataset reports the yield of peas from an agricultural
experiment conducted on six blocks. Shapiro--Wilk does not reject
normality of the pooled residuals and the Levene test does not reject
homoscedasticity, so \texttt{visstat()} applies Fisher's one-way ANOVA followed
by Tukey HSD post-hoc comparisons. The omnibus F-test is not significant
at \(\alpha = 0.05\), and all groups share the same compact-letter label.

\begin{verbatim}
anova_npk <- visstat(npk$block, npk$yield, conf.level = 0.95)
\end{verbatim}

Assumption diagnostics and box plots of pea yield by experimental block; compact letter display (TukeyHSD, $\alpha = 0.05$) shows no significant pairwise differences.}\label{fig:anova-example}

\subsubsection{Kruskal-Wallis rank sum test}\label{kruskal-wallis-rank-sum-test}

The \texttt{iris} dataset contains petal-width measurements for three species.
Here, the residual plots and very small Shapiro--Wilk and
Anderson--Darling \(p\)-values indicate non-normal residuals. Since
Shapiro--Wilk falls below \(\alpha\), \texttt{visstat()} switches to
\texttt{kruskal.test()} followed by Holm-adjusted \texttt{pairwise.wilcox.test()}. All
three species differ significantly in petal width, as indicated by
distinct compact-letter labels.

\begin{verbatim}
kruskal_iris <- visstat(iris$Species, iris$Petal.Width)
\end{verbatim}

\subsection{Both variables numeric}\label{both-variables-numeric}

\subsubsection{Simple linear regression}\label{simple-linear-regression}

\texttt{When\ both\ predictor\ and\ response\ are\ numerical,\ visstat()} checks the
standardised residuals from \texttt{lm()} graphically and with the
Shapiro--Wilk and Anderson--Darling tests; the Breusch--Pagan test
assesses homoscedasticity. Regardless of these diagnostics, a simple linear
regression fitted and displayed. The graphical shows
the fitted line \(\hat{y} = b_0 + b_1 x\), where \(b_0\) is the
intercept and \(b_1\) the slope, with pointwise confidence and prediction
bands at the specified \texttt{conf.level}.

The title
shows \(R^2\), the estimated regression
parameters with confidence intervals and \(p\)-values. The
returned object contains the regression statistics, the
residual-normality \(p\)-values, and pointwise estimates of the confidence
and prediction bands.

The \texttt{swiss} dataset records standardised fertility and socioeconomic
indicators for 47 French-speaking Swiss provinces in 1888. We examine
how the share of draftees achieving the highest army examination score
(\texttt{Examination}) predicts the fertility measure (\texttt{Fertility}), with
\texttt{conf.level\ =\ 0.99}. Both normality tests pass and the Breusch--Pagan
test confirms homoscedasticity, validating the linear model.

\begin{verbatim}
linreg_swiss <- visstat(swiss$Examination, swiss$Fertility, conf.level = 0.99)
\end{verbatim}

\begin{figure}
\includegraphics[width=0.48\linewidth]{visStatistics-RJournal_files/figure-latex/regression-example-1} \includegraphics[width=0.48\linewidth]{visStatistics-RJournal_files/figure-latex/regression-example-2} \caption{Simple linear regression of \texttt{Fertility} on \texttt{Examination} for the \texttt{swiss} dataset (\texttt{conf.level = 0.99}). Assumption diagnostics (Shapiro--Wilk, Anderson--Darling, Breusch--Pagan) and scatter plot with fitted regression line, 99\% confidence band (dark shading), and 99\% prediction band (light shading).}\label{fig:regression-example}
\end{figure}

\subsubsection{Spearman rank correlation}\label{spearman-rank-correlation}

For numeric--numeric input, correlation analysis is never triggered
automatically; it requires the explicit flag \texttt{correlation\ =\ TRUE}. With
wind speed (\texttt{Wind}) and ozone concentration (\texttt{Ozone}) from the
\texttt{airquality} dataset \citep{Chambers:2018}, a default \texttt{visstat()} call fits
simple linear regression but flags violated normality and
homoscedasticity. The warning recommends two paths: rerun with
\texttt{correlation\ =\ TRUE} within \texttt{visstat()}, or switch to a model outside
it. Staying within \texttt{visstat()}, \texttt{correlation\ =\ TRUE} gives an
assumption-free Spearman result at the cost of causal interpretability.

\begin{verbatim}
spearman_air <- visstat(airquality$Wind, airquality$Ozone, correlation = TRUE)
\end{verbatim}

\begin{figure}
\includegraphics[width=0.48\linewidth]{visStatistics-RJournal_files/figure-latex/spearman-example-1} \includegraphics[width=0.48\linewidth]{visStatistics-RJournal_files/figure-latex/spearman-example-2} \caption{Spearman rank correlation of \texttt{Wind} and \texttt{Ozone} from the \texttt{airquality} dataset (\texttt{correlation = TRUE}). Assumption diagnostics and scatter plot of ranked values annotated with $\rho$ and the $p$-value.}\label{fig:spearman-example}
\end{figure}

\subsection{Both variables categorical}\label{both-variables-categorical}

Observed frequencies are arranged in a contingency table, where rows
index the levels of the response and columns index the levels of the
predictor. \texttt{visstat()} tests the null hypothesis that the variables are
independent. If expected frequencies satisfy Cochran's rule, Pearson's
\(\chi^2\) test is used; otherwise, \texttt{visstat()} switches to Fisher's exact
test. For \(2 \times 2\) tables, Yates' continuity correction is applied
by default to Pearson's \(\chi^2\) test. The graphical output depends on
the selected test. General \(R \times C\) Pearson \(\chi^2\) tests show a
grouped column plot of row percentages with the \(p\)-value in the title,
followed by a mosaic plot with colour-coded Pearson residuals.
Yates-corrected \(2 \times 2\) \(\chi^2\) tests show the grouped column plot
only, because the corrected statistic is not decomposed into cell-level
residuals. Fisher's exact test shows absolute counts with \(N\) labels
above each bar and the \(p\)-value in the title.

The following examples are based on the \texttt{HairEyeColor} contingency
table, which is converted to the column-based data frame expected by
\texttt{visstat()} using \texttt{counts\_to\_cases()}.

\subsubsection{\texorpdfstring{Pearson's \(\chi^2\) test}{Pearson's \textbackslash chi\^{}2 test}}\label{pearsons-chi2-test}

With \texttt{Eye} and \texttt{Hair} from \texttt{HairEyeColor}, all expected cell counts
exceed the Cochran thresholds \citep{Cochran:1954}, so the \(\chi^2\)
approximation is used. For this \(4 \times 4\) table, \texttt{visstat()} returns
both a grouped bar chart of row percentages and a mosaic plot with tiles
coloured by Pearson residuals. The mosaic plot shows which cells drive
the association: blue tiles indicate observed counts above expectation
and red tiles indicate observed counts below expectation.

\begin{verbatim}
hair_eye_df <- counts_to_cases(as.data.frame(HairEyeColor))
visstat(hair_eye_df$Eye, hair_eye_df$Hair)
\end{verbatim}

\begin{figure}
\includegraphics[width=0.48\linewidth]{visStatistics-RJournal_files/figure-latex/chisq-example-1} \includegraphics[width=0.48\linewidth]{visStatistics-RJournal_files/figure-latex/chisq-example-2} \caption{Pearson's $\chi^2$ test applied to the \texttt{HairEyeColor} dataset. Grouped bar chart of eye colour by hair colour and mosaic plot with tiles coloured by Pearson residuals (blue: over-represented, red: under-represented).}\label{fig:chisq-example}
\end{figure}

\subsubsection{Fisher's exact test}\label{fishers-exact-test}

Restricting \texttt{HairEyeColor} to male participants with black or brown hair
and hazel or green eyes yields a \(2 \times 2\) table where one expected
frequency is less than 5, violating Cochran's rule \citep{Cochran:1954}.
\texttt{visstat()} therefore applies Fisher's exact test. For \(2 \times 2\)
tables, the returned object also contains the conditional maximum
likelihood estimate of the odds ratio and its confidence interval.

\begin{verbatim}
hair_eye_male <- HairEyeColor[, , 1]
black_brown_hazel_green <- hair_eye_male[1:2, 3:4]
black_brown_hazel_green_df <- counts_to_cases(
  as.data.frame(black_brown_hazel_green))
fisher_stats <- visstat(black_brown_hazel_green_df$Eye,
                        black_brown_hazel_green_df$Hair)
fisher_stats$estimate   # odds ratio
\end{verbatim}

\begin{verbatim}
#> odds ratio 
#>  0.5062015
\end{verbatim}

\begin{verbatim}
fisher_stats$conf.int   # 95 % CI
\end{verbatim}

\begin{verbatim}
#> [1] 0.07725895 2.40885255
#> attr(,"conf.level")
#> [1] 0.95
\end{verbatim}

Bar chart of absolute counts with the $p$-value; the odds ratio and confidence interval are accessible from the returned object.}\label{fig:fisher-exact-example}

\subsection{Ordered factors}\label{ordered-factors-1}

When the response is ordered and the predictor is categorical,
\texttt{visstat()} converts the response to integer level codes and redirects
to the non-parametric Wilcoxon or Kruskal--Wallis path above. When both
variables are ordered factors and \texttt{correlation\ =\ TRUE}, \texttt{visstat()}
instead tests for a monotone association via Kendall's \(\tau_b\)
\citep{Kendall:1945, Agresti:2010}. The Kendall output is a single jittered
rank--rank scatter plot that visualises the monotone trend, colour-codes
points by the predictor level, and annotates the plot with \(\tau_b\) and
the \(p\)-value.

\subsubsection{\texorpdfstring{Kendall's \(\tau_b\)}{Kendall's \textbackslash tau\_b}}\label{kendalls-tau_b}

A hypothetical survey of 150 secondary-school students records alcohol
consumption frequency and academic performance on five-point ordinal
scales. A negative monotone association is induced by construction:
students who consume alcohol more frequently tend to have lower academic
performance.

\begin{verbatim}
set.seed(42)
n <- 150
xs <- sample(1:5, n, replace = TRUE)
ys <- pmin(5, pmax(1, (6 - xs) + sample(-1:1, n, replace = TRUE)))
likert_alc  <- c("never", "rarely", "sometimes", "often", "always")
likert_perf <- c("poor",  "fair",   "ok",        "good",  "great")
alcohol     <- ordered(likert_alc[xs],  levels = likert_alc)
performance <- ordered(likert_perf[ys], levels = likert_perf)
kendall_result <- visstat(performance, alcohol, correlation = TRUE)
\end{verbatim}

\section{Saving graphical output}\label{saving-graphical-output}

All plots can be saved in any format supported by \CRANpkg{Cairo}
\citep{Urbanek:2025} --- \texttt{"png"}, \texttt{"pdf"}, \texttt{"svg"}, \texttt{"ps"}, \texttt{"tiff"} --- by
supplying the \texttt{graphicsoutput} and \texttt{plotDirectory} arguments. File names
follow the pattern \texttt{testname\_namey\_namex.ext} by default, or a
user-supplied \texttt{plotName}. The full file paths of generated graphics are
stored as the \texttt{"plot\_paths"} attribute on the returned \texttt{"visstat"}
object.

\section{Limitations}\label{limitations}

\subsection{Scope of automated test selection}\label{scope-of-automated-test-selection}

All tests are applied with their default settings from base R. As a
consequence, paired tests are not supported and interaction terms are
not available in the ANOVA path. Only simple linear regression is
implemented, as multiple regression cannot be visualised in two
dimensions. When regression assumptions are violated,
\CRANpkg{visStatistics} offers Spearman rank correlation
(\texttt{correlation\ =\ TRUE}) as the sole automated alternative. Alternative
methods such as data transformations, generalised linear models, or
robust regression are not implemented: each requires user judgment ---
about the transformation family, the link function, or the estimator ---
that cannot be automated without substantially expanding the decision
tree and increasing the risk of Type I error inflation.

\subsection{P-value-based assumption testing}\label{p-value-based-assumption-testing}

No single test maintains optimal Type I error rates and power across all
distributions \citep{Olejnik:1987}. Normality tests behave poorly at both
extremes of the sample-size range: they fail to detect non-normality in
small samples and flag negligible departures as significant in large
ones \citep{Ghasemi:2012, Fagerland:2012}. The \(n > 50\) per-group threshold
for bypassing the Shapiro--Wilk test is necessarily arbitrary;
simulation studies indicate that the required sample size for the
central limit theorem to take effect depends on the skewness and
kurtosis of the underlying distribution \citep{Fagerland:2012}. Assumption
tests provide no information on the nature of the deviation
\citep{Shatz:2024}, and combining multiple tests inflates the overall Type I
error rate. For these reasons, the diagnostic plots generated by
\texttt{visstat()} should always be inspected alongside the automated decision;
the \texttt{conf.level} argument provides a direct handle on the \(\alpha\) used
throughout.

\subsection{Visualisation}\label{visualisation}

The package depends exclusively on base R graphics with no
\CRANpkg{ggplot2} \citep{Wickham:2016} dependency, keeping the transitive
dependency footprint minimal. For more polished, annotated plots of
chosen statistical tests, see \CRANpkg{ggstatsplot} \citep{Patil:2021} or
\CRANpkg{ggpubr} \citep{Kassambara:2026}.

\section{Summary}\label{summary}

\CRANpkg{visStatistics} provides a single-function interface to
automated statistical test selection, execution, and visualisation for
the most commonly used inferential frameworks in two-variable analyses.
Its principal contribution is the integration of the full pipeline ---
data-type detection, distributional assumption checking on GLM
residuals, test routing, and annotated graphical output --- into a
reproducible, server-deployable workflow with a minimal dependency
footprint. The package is available on CRAN at
\texttt{https://CRAN.R-project.org/package=visStatistics} and on GitHub at
\texttt{https://github.com/shhschilling/visStatistics}.

\section{Appendix A: The general linear model}\label{glm}

The general linear model \citep{Searle:1971} provides a unified framework
underlying Student's t-test, Fisher's ANOVA, and simple linear
regression:

\[Y_i = \beta_0 + \beta_1 x_{i1} + \cdots + \beta_{k-1} x_{i,k-1} + \varepsilon_i,
\quad \varepsilon_i \sim N(0, \sigma^2).\]

\textbf{Student's t-test} uses one binary indicator variable \(x_{i1}\);
testing \(H_0: \beta_1 = 0\) is equivalent to testing
\(H_0: \mu_1 = \mu_2\).

\textbf{Fisher's ANOVA} uses \(k-1\) indicator variables for \(k\) groups;
testing \(H_0: \beta_1 = \cdots = \beta_{k-1} = 0\) is equivalent to
testing equality of all group means.

\textbf{Simple linear regression} uses one continuous predictor; testing
\(H_0: \beta_1 = 0\) examines whether a linear relationship exists.

The two-sample case connects t-tests and ANOVA: \(t^2 = F\). For two
groups, \texttt{t.test(var.equal\ =\ TRUE)} and \texttt{aov()} yield identical
\(p\)-values, as do \texttt{t.test(var.equal\ =\ FALSE)} and \texttt{oneway.test()}.

\section{Appendix B: Test statistics}\label{tests}

\subsection{Normality tests}\label{normality-tests}

Both normality tests are applied to the internally studentised residuals
\(r_i^* = e_i / (s\sqrt{1-h_{ii}})\), where \(e_i\) are the raw residuals,
\(s^2\) is the residual mean square, and \(h_{ii}\) is the leverage of
observation \(i\) \citep{Cook:1982}.

\subsubsection{\texorpdfstring{Shapiro--Wilk test \texttt{shapiro.test()}}{Shapiro--Wilk test shapiro.test()}}\label{shapirowilk-test-shapiro.test}

The Shapiro--Wilk test \citep{Shapiro:1965} evaluates whether a sample
\(x_1, \ldots, x_n\) comes from a normal distribution. Let \(x_{(i)}\)
denote the \(i\)-th order statistic. The test statistic is

\[W = \frac{\left(\displaystyle\sum_{i=1}^{n} a_i\, x_{(i)}\right)^2}
           {\displaystyle\sum_{i=1}^{n} (x_i - \bar{x})^2},\]

where \(\mathbf{a} = (a_1,\ldots,a_n)^\top\) is the normalised coefficient
vector,

\[\mathbf{a} =
\frac{V^{-1}\mathbf{m}}
{\sqrt{\mathbf{m}^\top V^{-1}V^{-1}\mathbf{m}}},\]

and \(Z_{(1)},\ldots,Z_{(n)}\) are the ordered values expected from a
sample of size \(n\) drawn from a standard normal distribution. The vector
\(\mathbf{m}\) contains their expected positions, and \(V\) describes how
these ordered positions vary together across repeated standard normal
samples \citep{Shapiro:1965}. \(W\) takes values in \((0, 1]\); values close to 1
indicate normality. Simulation studies show that the Shapiro--Wilk test
has the highest power among normality tests for small to moderate sample
sizes (\(n = 10\) to \(100\)) \citep{Razali:2011}. It drives the
parametric/non-parametric branching decision in \texttt{visstat()}; for groups
with more than 50 observations normality is assumed by the central limit
theorem \citep{Lumley:2002, Rasch:2011}.

\subsubsection{\texorpdfstring{Anderson--Darling test \texttt{ad.test()}}{Anderson--Darling test ad.test()}}\label{andersondarling-test-ad.test}

The Anderson--Darling test \citep{Anderson:1952} is particularly sensitive to
deviations in the tails of the distribution \citep{Razali:2011, Yap:2011}.
Let \(z_i = (x_{(i)} - \bar{x})/s\) be the standardised order statistics
and \(\Phi\) the standard normal CDF. The test statistic is

\[A^2 = -n - \frac{1}{n}\sum_{i=1}^{n}(2i-1)
        \left[\ln\Phi(z_i) + \ln\!\left(1 - \Phi(z_{n+1-i})\right)\right].\]

The implementation uses \texttt{ad.test()} from \CRANpkg{nortest}
\citep{Gross:2015}. The Anderson--Darling result is shown in all assumption
plots but does \textbf{not} drive the branching decision; only Shapiro--Wilk
does.

\subsection{Homoscedasticity tests}\label{homoscedasticity-tests}

\subsubsection{\texorpdfstring{The Levene--Brown--Forsythe test \texttt{levene.test()}}{The Levene--Brown--Forsythe test levene.test()}}\label{the-levenebrownforsythe-test-levene.test}

The test \citep{Brown:1974} improves upon Levene's original proposal
\citep{Levene:1960} by centring on the group median rather than the group
mean, making it robust to skewed data and outliers \citep{Allingham:2012}.
The implementation \texttt{levene.test()} mimics the default behaviour of
\texttt{leveneTest()} in \CRANpkg{car} \citep{Fox:2019} and is the sole driver of
the parametric/Welch branching decision.

For each observation \(y_{ij}\) in group \(i\), the absolute deviation from
the group median \(\tilde{y}_i\) is

\[z_{ij} = |y_{ij} - \tilde{y}_i|.\]

The test statistic is the \(F\)-statistic from a one-way ANOVA on the
\(z_{ij}\) values:

\[F = \frac{(N-k)\displaystyle\sum_{i=1}^{k} n_i (\bar{z}_i - \bar{z})^2}
           {(k-1)\displaystyle\sum_{i=1}^{k}\sum_{j=1}^{n_i}(z_{ij} - \bar{z}_i)^2},\]

where \(k\) is the number of groups, \(N = \sum_{i=1}^k n_i\) is the total
sample size, \(n_i\) is the sample size of group \(i\), \(\bar{z}_i\) is the
mean of the absolute deviations within group \(i\), and \(\bar{z}\) is the
overall mean of all absolute deviations. Under the null hypothesis of
equal variances the statistic follows \(F(k-1, N-k)\).

\subsubsection{\texorpdfstring{Bartlett's test \texttt{bartlett.test()}}{Bartlett's test bartlett.test()}}\label{bartletts-test-bartlett.test}

Bartlett's test \citep{Bartlett:1937} has greater power than the
Brown--Forsythe test when normality holds \citep{Allingham:2012}. Its test
statistic is

\[K^2 = \frac{(N-k)\ln s_p^2 - \displaystyle\sum_{i=1}^k (n_i-1)\ln s_i^2}
{1 + \dfrac{1}{3(k-1)}\!\left(\displaystyle\sum_{i=1}^k \frac{1}{n_i-1} - \frac{1}{N-k}\right)},\]

where \(s_i^2\) is the sample variance of group \(i\) and \(s_p^2\) is the
pooled variance

\[s_p^2 = \frac{1}{N-k}\sum_{i=1}^k (n_i-1)\,s_i^2.\]

Under the null hypothesis the statistic approximately follows
\(\chi^2(k-1)\). Bartlett's result is shown in the assumption plot but
does \textbf{not} drive the branching decision; only \texttt{levene.test()} does.

\subsubsection{\texorpdfstring{Breusch--Pagan test \texttt{bp.test()}}{Breusch--Pagan test bp.test()}}\label{breuschpagan-test-bp.test}

For simple linear regression, group-based variance tests are not
applicable. The Breusch--Pagan test \citep{Breusch:1979} assesses whether the
variance of the residuals from the regression model depends on the
predictor values. Its result is displayed in the regression assumption
plot.

\subsection{Student's t-test and Welch's t-test}\label{students-t-test-and-welchs-t-test}

The test statistic for Student's t-test (\texttt{t.test(var.equal\ =\ TRUE)}) is

\[t = \frac{\bar{x}_1 - \bar{x}_2}{s_p \sqrt{\dfrac{1}{n_1} + \dfrac{1}{n_2}}},\]

where \(\bar{x}_1\) and \(\bar{x}_2\) are the sample means, \(n_1\) and \(n_2\)
the sample sizes, and \(s_p\) the pooled standard deviation

\[s_p = \sqrt{\frac{(n_1-1)s_1^2 + (n_2-1)s_2^2}{n_1+n_2-2}},\]

with \(s_1^2\) and \(s_2^2\) the sample variances. The statistic follows a
\(t\)-distribution with \(\nu = n_1 + n_2 - 2\) degrees of freedom.

Welch's t-test relaxes the homoscedasticity assumption. Its statistic is

\[t = \frac{\bar{x}_1 - \bar{x}_2}{\sqrt{s_1^2/n_1 + s_2^2/n_2}},\]

with degrees of freedom approximated by the Welch--Satterthwaite
equation \citep{Welch:1947, Satterthwaite:1946}:

\[\nu \approx \frac{\left(\dfrac{s_1^2}{n_1} + \dfrac{s_2^2}{n_2}\right)^2}
{\dfrac{(s_1^2/n_1)^2}{n_1-1} + \dfrac{(s_2^2/n_2)^2}{n_2-1}}.\]

Welch's methods outperform their classical counterparts when variances
differ \citep{Moser:1992, Fagerland:2009, Delacre:2017}.

\subsection{Wilcoxon rank-sum test}\label{wilcoxon-rank-sum-test-1}

The two-level factor \texttt{x} defines two groups with sample sizes \(n_1\) and
\(n_2\). All \(N = n_1 + n_2\) observations are pooled and assigned ranks
\(1\) to \(N\). Let \(W_1 = \sum_{i=1}^{n_1} R(x_{1,i})\) denote the rank sum
of the first group, where \(R(x_{1,i})\) is the rank of observation
\(x_{1,i}\) in the pooled sample. The test statistic returned by
\texttt{wilcox.test()} \citep{Mann:1947} is the Mann--Whitney \(U\) statistic of the
first group:

\[W = U_1 = W_1 - \frac{n_1(n_1+1)}{2}.\]

An exact \(p\)-value is computed when both groups contain fewer than 50
observations and the data contain no ties; otherwise a normal
approximation with continuity correction is used.

\subsection{Fisher's one-way ANOVA and Welch's heteroscedastic ANOVA}\label{fishers-one-way-anova-and-welchs-heteroscedastic-anova}

Fisher's ANOVA test statistic \citep{Fisher:1990} is

\[F = \frac{MS_\text{between}}{MS_\text{within}}
   = \frac{SS_\text{between}/(k-1)}{SS_\text{within}/(N-k)}
   = \frac{\displaystyle\sum_{i=1}^{k} n_i (\bar{x}_i - \bar{x})^2\;/\;(k-1)}
          {\displaystyle\sum_{i=1}^{k}\sum_{j=1}^{n_i}(x_{ij}-\bar{x}_i)^2\;/\;(N-k)},\]

where \(k\) is the number of groups, \(N = \sum_{i=1}^k n_i\) is the total
sample size, \(\bar{x}_i\) is the mean of group \(i\), \(\bar{x}\) is the
overall mean, and \(x_{ij}\) is observation \(j\) in group \(i\). Under the
null hypothesis \(F \sim F(k-1, N-k)\). Post-hoc comparisons use
\texttt{TukeyHSD()} \citep{Hochberg:1987}, which controls the family-wise error rate
via the studentised range distribution \citep{Abdi:2007}.

Welch's heteroscedastic ANOVA (\texttt{oneway.test()}) \citep{Welch:1951}
down-weights groups with large variance. Its test statistic is

\[F_W = \frac{\displaystyle\sum_{i=1}^{k} w_i (\bar{y}_i - \bar{y}_w)^2\;/\;(k-1)}
{1 + \dfrac{2(k-2)}{k^2-1} \displaystyle\sum_{i=1}^{k} \dfrac{(1-w_i/w)^2}{n_i-1}},\]

where \(w_i = n_i/s_i^2\) are the inverse-variance weights,
\(w = \sum_{i=1}^{k} w_i\), and
\(\bar{y}_w = \sum_{i=1}^{k} w_i \bar{y}_i / w\) is the weighted grand
mean. Degrees of freedom are adjusted via a Satterthwaite-type
approximation \citep{Satterthwaite:1946}. Post-hoc comparisons use
\texttt{games.howell()} \citep{Games:1976}, which applies separate variance
estimates per pair. Compact letter displays are produced by
\texttt{multcompLetters()} from \CRANpkg{multcompView} \citep{Graves:2026}.

\subsection{Kruskal--Wallis test}\label{kruskalwallis-test}

When normality is rejected, \texttt{kruskal.test()} \citep{Kruskal:1952} tests
whether the groups come from the same population based on ranks:

\[H = \frac{12}{N(N+1)} \sum_{i=1}^{k} n_i \left(\bar{R}_i - \bar{R}\right)^2,\]

where \(n_i\) is the sample size of group \(i\), \(k\) is the number of
groups, \(\bar{R}_i\) is the average rank of group \(i\), \(N\) is the total
sample size, and \(\bar{R} = (N+1)/2\) is the expected average rank under
the null hypothesis. The statistic approximately follows \(\chi^2(k-1)\).
Post-hoc comparisons use Holm-adjusted \texttt{pairwise.wilcox.test()}
\citep{Holm:1979}.

\subsection{\texorpdfstring{Pearson's \(\chi^2\) test and Fisher's exact test}{Pearson's \textbackslash chi\^{}2 test and Fisher's exact test}}\label{pearsons-chi2-test-and-fishers-exact-test}

Let \(O_{ij}\) and \(E_{ij}\) denote the observed and expected frequencies
in row \(i\) and column \(j\) of an \(R \times C\) contingency table, where
rows index the \(R\) levels of the response \(y\) and columns the \(C\) levels
of the predictor \(x\). The Pearson residual for cell \((i,j)\) is

\[r_{ij} = \frac{O_{ij} - E_{ij}}{\sqrt{E_{ij}}}, \quad i = 1,\ldots,R,\quad j = 1,\ldots,C.\]

The test statistic of Pearson's \(\chi^2\) test \citep{Pearson:1900} is

\[\chi^2 = \sum_{i=1}^{R}\sum_{j=1}^{C} r_{ij}^2
         = \sum_{i=1}^{R}\sum_{j=1}^{C} \frac{(O_{ij}-E_{ij})^2}{E_{ij}},\]

compared to \(\chi^2\!\left((R-1)(C-1)\right)\). For \(2\times 2\) tables,
Yates' continuity correction \citep{Yates:1934} is applied by default. For
general \(R \times C\) tables, \texttt{visstat()} supplements the bar chart with
a mosaic plot in which tiles are coloured by \(r_{ij}\) (blue: positive,
red: negative).

Fisher's exact test \citep{Fisher:1970} is applied when Cochran's rule
\citep{Cochran:1954} is violated. It evaluates the exact hypergeometric
probability of the observed table given the fixed margins. For
\(2 \times 2\) tables with row totals \(n_{i\cdot}\) and column totals
\(n_{\cdot j}\), this probability is

\[\mathbb{P}(N \mid n_{1\cdot},n_{2\cdot},n_{\cdot 1},n_{\cdot 2})
= \frac{\dbinom{n_{1\cdot}}{n_{11}}\dbinom{n_{2\cdot}}{n_{21}}}{\dbinom{n}{n_{\cdot 1}}},\]

where \(n = n_{1\cdot}+n_{2\cdot}\) is the total sample size. For
\(2\times 2\) tables, \texttt{fisher.test()} additionally returns the conditional
maximum likelihood estimate of the odds ratio
\(\widehat{\text{OR}} = n_{11}n_{22}/(n_{12}n_{21})\) and its confidence
interval.

\subsection{Spearman rank correlation}\label{spearman-rank-correlation-1}

For two numeric variables with \texttt{correlation\ =\ TRUE}, \texttt{visstat()} calls
\texttt{cor.test(x,\ y,\ method\ =\ "spearman")} to measure the monotonic
association between \(x\) and \(y\) using ranks, evaluating linear
dependence on the ranked data rather than on the original scale.
Spearman's \(\rho\) is Pearson's \(r\) applied to the ranks:

\[\rho = r\!\left(\operatorname{rank}(x),\, \operatorname{rank}(y)\right)
= \frac{\displaystyle\sum_{i=1}^n(\operatorname{rank}(x_i)-\overline{\operatorname{rank}(x)})
                                 (\operatorname{rank}(y_i)-\overline{\operatorname{rank}(y)})}
       {\sqrt{\displaystyle\sum_{i=1}^n(\operatorname{rank}(x_i)-\overline{\operatorname{rank}(x)})^2}
        \sqrt{\displaystyle\sum_{i=1}^n(\operatorname{rank}(y_i)-\overline{\operatorname{rank}(y)})^2}}.\]

For inference, \texttt{cor.test(...,\ method\ =\ "spearman")} computes an exact
\(p\)-value for small samples without ties by evaluating all \(n!\) rank
permutations. For larger samples or when ties are present, it uses an
approximation to the null distribution of the test statistic. No
distributional assumptions on the original data are required.

\subsection{\texorpdfstring{Kendall's \(\tau_b\)}{Kendall's \textbackslash tau\_b}}\label{kendalls-tau_b-1}

For two ordinal variables with \(n\) joint observations, let \(C\) denote
the number of concordant pairs and \(D\) the number of discordant pairs.
Kendall's \(\tau_b\) \citep{Kendall:1945} is

\[\tau_b = \frac{C - D}{\sqrt{(n_0 - n_1)(n_0 - n_2)}},\]

where \(n_0 = n(n-1)/2\), \(n_1 = \sum_i t_i(t_i-1)/2\) sums over groups of
tied ranks in the response (with \(t_i\) the size of the \(i\)-th tie
group), and \(n_2\) is the analogous quantity for the predictor. The
denominator correction ensures \(\tau_b \in [-1, 1]\) even in the presence
of ties, which Spearman's \(\rho\) does not guarantee \citep{Kendall:1945}.
\(\tau_b\) is therefore preferred for ordinal data with few levels, where
ties are unavoidable \citep{Agresti:2010, Xu:2013}.
