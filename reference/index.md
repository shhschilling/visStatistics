# Package index

## All functions

- [`counts_to_cases()`](https://shhschilling.github.io/visStatistics/reference/counts_to_cases.md)
  : Convert data frame of counts to dataframe of cases. The data frame
  must contain a column with frequencies (counts) as generated by
  as.data.frame from a contingency table
- [`effect_size()`](https://shhschilling.github.io/visStatistics/reference/effect_size.md)
  : Compute an effect-size estimate for a visstat result
- [`openGraphCairo()`](https://shhschilling.github.io/visStatistics/reference/openGraphCairo.md)
  : Cairo wrapper function with plot capture capability
- [`saveGraphVisstat()`](https://shhschilling.github.io/visStatistics/reference/saveGraphVisstat.md)
  : Saves Graphical Output with plot capture capability
- [`visstat()`](https://shhschilling.github.io/visStatistics/reference/visstat.md)
  : Wrapper for visstat_core Allowing Three Different Input Styles
